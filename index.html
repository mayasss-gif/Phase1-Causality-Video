Skeleton Construction
Step 1: SNP Source Nodes
Union of GWAS + eQTL SNPs. Each typed as GWAS_Catalog, eQTL_Study, or both. Marked is_anchor=True for directionality enforcement.
Step 2: Gene Regulatory Nodes
All genes from gene_list (DEG + GWAS + MR union). 20+ attributes per gene: expression, ACE, essentiality, alignment, causal_tier.
Step 3: Program Nodes
REACTOME/KEGG/WikiPathway + GO_BP + cell types from Bisque deconvolution. Ontology: Main_Class / Sub_Class hierarchy.
Step 4: Disease Trait Node
Single node: Disease_Activity. Layer='trait', type='clinical_outcome'. Terminal sink in the causal DAG topology.
Step 5: Edge Construction
Co-expression: |r|>0.4 with STATISTICAL weight. Pathway membership: gene→pathway (DATABASE=0.80). Cell markers: deconvolution signature.
Confidence Formula
scaled = (|r| - 0.4) / (1.0 - 0.4)
base_prob = min(0.99, scaled × W['STATISTICAL'])
Both directions (u→v, v→u) added

Phenotype Arbitration: Drivers vs. Reactive Biomarkers

Does a gene CAUSE disease, or does disease CAUSE the gene's expression change? This determines edge direction.

CAUSAL DRIVER: Gene → Disease
Criteria (ANY):
• is_gwas_hit = True
• Has MR evidence
• perturbation_ace ≤ -0.1

Edge properties:
• edge_type = 'phenotype_driver'
• mechanism = 'causal_disease_driver'
• evidence: 'causal_anchor_supported'

Reverse edge (Disease→Gene) is REMOVED
REACTIVE BIOMARKER: Disease → Gene
Criteria:
• Correlates with disease BUT
• No GWAS, no MR, ACE > -0.1
• Expression changes are CONSEQUENCE

Edge properties:
• edge_type = 'reactive_biomarker'
• mechanism = 'disease_induced_expression'
• evidence: 'lacks_causal_anchors'

Forward edge (Gene→Disease) is REMOVED
Prior Injection Pipeline
MR Evidence Injection
For each gene with MR results:
• mechanism based on beta direction
• P-value scaling: p<0.001→1.0, p<0.01→0.90, else 0.80
• final_prob = W['MR'](0.95) × scalar
• REMOVES reverse edge
• Adds 'mendelian_randomization_causality'
eQTL Edge Injection
For each SNP-Gene pair:
• base_prob = W['EQTL'](0.85) × min(1, |beta|)
• If gene in coloc_proxy (GWAS∩eQTL):
  Bayesian_agg(base, W['GWAS']) → massive boost
• Creates SNP→Gene edges
• Evidence: 'eqtl_study_support'
SIGNOR Interaction Injection
For each curated physical interaction:
• New edge: confidence=0.1 (seed)
• Existing edge: ADD 'signor_physical_interaction'
• Full SIGNOR boost applied later in constraint #5
Granger Causality Injection
Significant pairs (q<0.05):
• base_prob = W['STAT'] × min(1, F/40)
• ONE-NODE RESCUE (q<0.01):
  Missing partner recovered from skeleton
5 Advanced Constraint Rules
1. Genetic Anchor Flow
GWAS root → non-root: BOOST confidence with W['GWAS']×0.8.
Reverse edge: PENALIZE conf×(1-W['GWAS']×0.5).
Mark reverse as 'potential_feedback_loop'.
2. Hybrid Temporal Arbitration
Granger + impulse: REMOVE reverse, 'spatiotemporal_causality_verified'.
Granger only: BOOST + penalize reverse.
Impulse only: REMOVE reverse.
3. Perturbation Asymmetry
ACE_u ≤ -0.1 AND ACE_v ≥ -0.1 AND |Δ| ≥ 0.2:
Enforce u→v direction, REMOVE reverse.
mechanism='fitness_driver_regulation'
dynamic_w = W['CRISPR']×min(1,|ACE|).
4. Cellular Context Veto
Different cell types:
If ligand-receptor pair: RETYPE to 'paracrine_signaling'.
Otherwise: REMOVE edge + reverse (intracellular only).
5. SIGNOR Arbitration Bonus
Curated SIGNOR interaction:
BOOST: Bayesian_agg(current, 0.90).
REMOVE reverse edge (curated direction wins).
Strongest physical evidence override.

MRValidator: Instrument Selection
MODULE 2  |  MENDELIAN RANDOMIZATION VALIDATION
prepare_instruments() Pipeline
# 1. Filter by gene gene_data = eqtl[eqtl.gene == target] # 2. Genome-wide significance gene_data = gene_data[pvalue < 5e-8] # 3. Compute instrument strength r2 = β² / (β² + se² × n) F = r2 × (n-2) / (1-r2) # 4. Strong instrument gate KEEP IF F ≥ 10 # 5. Sort by F-statistic instruments.sort(F, descending) # 6. LD pruning (r² < 0.001) FOR each: check independence from all previously kept

5 Mendelian Randomization Methods
IVW
Inverse Variance Weighted

wi = 1/sey²
β = Σ(w·bx·by) / Σ(w·bx²)
se = √(1/Σ(w·bx²))

Primary method. Assumes all instruments valid. Min: 3.
MR-Egger
Weighted Regression

by = α + β·bx
W = diag(1/se²)
β̂ = (X'WX)⁻¹X'Wy

Intercept≠0 detects pleiotropy. InSIDE assumption. Min: 4.
Weighted Median
Robust Median

ratios = by/bx
weights = bx²/se²
β = weighted median

Valid if ≥50% weight from valid IVs. Bootstrap SE (1000).
Weighted Mode
Kernel Density

h = 0.9·σ·n⁻⁰·²
density = Σ(w·N(grid|r,h))
β = grid[argmax]

Valid if largest cluster is valid. Bootstrap SE (1000).
MR-PRESSO
Outlier Detection

RSS = Σ(w·(by-β·bx)²)
1000 permutations
p = mean(null≥obs)

Identifies outlier IVs (|resid|>3·se). Re-runs IVW clean.
4 Sensitivity Analyses
Cochran's Q Heterogeneity
Tests if instrument-specific estimates are homogeneous.

Q = Σ wi(ratioi − βIVW)²
p = 1 − χ²cdf(Q, df=n−1)
I² = max(0, (Q−df)/Q × 100)

Heterogeneity detected if p < 0.05
Steiger Directionality
Confirms causal direction: exposure → outcome.

zexp = arctanh(√r²exp)
zout = arctanh(√r²out)
zstat = (zexp − zout) / √(2/997)

CRITICAL: Must pass for validation
Leave-One-Out
Checks if any single instrument drives the result.

FOR i in 1..n:
  subset = instruments \ {i}
  run_ivw(subset) → β, se, p

Dramatic change → instrument may be invalid. Min: 4.
Egger Intercept
Tests for directional pleiotropy.

H₀: intercept = 0 (no pleiotropy)
H₁: intercept ≠ 0 (pleiotropy)

p < 0.05 → IVW biased, use Egger slope instead.
MR Validation Decision Logic
validate_edge() Decision Algorithm
# 1. Prepare instruments instruments = prepare_instruments(gene) IF len < 3: RETURN validated=False # 2. Run all 5 MR methods ivw = run_ivw(instruments) egger = run_egger(instruments) wm = run_weighted_median(instruments) mode = run_weighted_mode(instruments) presso= run_mr_presso(instruments) # 3. Run sensitivity tests cochran = run_cochran_q(ivw.beta) steiger = run_steiger_test() loo = run_leave_one_out() # 4. Count significant methods methods_sig = count(p < 0.05) dir_consistent = all same sign # 5. Decision gates VALIDATED = ivw.p < 0.05 AND methods_sig ≥ 2 AND steiger.correct_direction
VALIDATED = True
✓ IVW p-value < 0.05
✓ At least 2 of 4 methods significant
✓ Steiger confirms correct direction
SENSITIVITY PASSED = True
✓ No heterogeneity (Cochran Q p≥0.05)
✓ No pleiotropy (Egger intercept p≥0.05)
✓ Direction consistent (all same sign)
✓ Steiger confirms direction
